Visual Studio Code

Co je Visual Studio Code?

Visual Studio Code (VS Code) je bezplatný, open-source editor kódu vyvinutý společností Microsoft. Je navržený pro vývojáře a podporuje širokou škálu programovacích jazyků, jako jsou JavaScript, Python, C++, HTML, CSS a mnoho dalších. VS Code nabízí funkce, jako je zvýrazňování syntaxe, automatické dokončování kódu (IntelliSense), integrovaný terminál, a debugování. Díky rozšiřitelným pluginům můžete přidávat nové funkce a nástroje, což ho činí velmi flexibilním pro různé typy projektů a technologií. Je dostupný pro Windows, macOS i Linux.
