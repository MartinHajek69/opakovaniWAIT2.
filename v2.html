Co je Github

GitHub je platforma pro vývojáře, která umožňuje hostování a správu verzí kódu pomocí systému Git. Umožňuje spolupráci na projektech, sdílení kódu, správu změn a vytváření open-source i soukromých projektů. GitHub je velmi populární mezi programátory díky funkcím, jako jsou pull requesty, issues, a možnost snadného sledování historie úprav kódu.
